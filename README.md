#🚀 LinkedIn Post Generator (AI‑Powered)

An AI‑powered tool that generates high‑quality, professional LinkedIn posts using LangChain + Groq LLM with smart prompt engineering and a Streamlit user interface.

🧠 Overview

This project leverages a large language model to produce LinkedIn posts tailored to your selected:

Topic

Post Length (Short, Medium, Long)

It uses few‑shot learning by retrieving example posts from your dataset to guide the writing style and structure, producing better, context‑aware output.

✨ Features

✔ Interactive Streamlit web app
✔ Few‑shot learning for style consistency
✔ User‑controlled topic & length
✔ Professional English content only
✔ Modular, clean architecture

🛠 Technology Stack

Python

Streamlit – UI

LangChain – Prompt + parsing

Groq LLM – Fast inference

JSON & pandas – Data processing
